🦁 Django 2주차 세션 정리

1. 개발 환경 준비

  • 프로젝트 구조: likelion_13th_session
  • 터미널 명령어: pipenv shell
  • 장고 설치: pip install django
  • 가상 환경 관리: pipenv로 python 패키지 관리

2. 프로젝트 시작 흐름

  • django-admin startproject → 프로젝트 생성
  • python manage.py startapp → 앱 생성
  • settings.py에서 앱 등록
  • views.py에서 함수 작성
  • urls.py에서 라우팅 설정
  • templates/에서 HTML 파일 작성

3. HTML 템플릿 예시 구성

  • 배경색: #111, 텍스트: #fff
  • Navbar 구성: display: flex로 메뉴 정렬
  • context로 전달된 값 출력

4. context 전달 & render 사용

  • views.py에서 딕셔너리 형태로 데이터 전달
  • render(request, 'template.html', context)
  • HTML에서 {{ key }} 문법으로 출력
